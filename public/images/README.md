# Images Directory

Para o efeito funcionar corretamente, você precisa adicionar as seguintes imagens nesta pasta:

1. **hero-off.png** - Imagem base (sem capacete)
2. **hero-on.png** - Imagem com efeito (com capacete)
3. **inspired-by-lando-norris.png** - Logo/créditos (120x120px ou similar)

As duas primeiras imagens devem ter a mesma resolução e dimensões para o efeito funcionar corretamente.

O efeito criará uma máscara interativa que revela a imagem "hero-on.png" quando você move o mouse sobre a tela.
